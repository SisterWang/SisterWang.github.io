# Promise

Promise是ES6中用来解决异步回调问题的方案。

# 状态

Promise的英文直译过来就是“承诺”的意思。其拥有三种状态，且状态一旦从等待变为其他两种之一，状态将不能更改：

  • Pending（等待）
  • Fulfilled（成功）
  • Rejected（失败）

举一个栗子：

我承诺女朋友，要在她生日当天送她一个名牌包包。
这时，对于我女朋友来说，我的承诺已经成立，此时这个承诺处于等待状态。
如果我在生日当天确实送了包包，那说明这个承诺已经成功；反之，如果我忘记了，那这个承诺即变为失败。这个状态此时已经不能改变，送了就是送了，没送就是没送。你可以选择事后补救，但这并不影响当前承诺的结果。

# 基本使用

可以通过new关键字来创建Promise对象，并为其传递一个函数。该函数接收两个参数：

  1. resolve：调用运行该方法，可以将Promise的状态修改为Fulfilled
  2. reject：调用该方法，可以将Promise的状态修改为Rejected

Promise对象的then方法接收状态变为成功时的回调，其参数为resolve方法的参数；catch方法用于错误的回调。

new Promise(function (resolve, reject) {
    resolve() // 成功
    reject() // 失败
}).then(res => {
    // ...
}).catch(err => {
    // ...
})

# 链式调用

为了解决传统回调地狱问题，then方法依然会返回新的Promise对象，该对象依然可以使用then方法。这样的设计可以方便进行链式调用：

// then 方法中的内容从上到下依次执行
new Promise(function (resolve, reject) {
    resolve()
}).then(res => {
    // ...
}).then(res => {
    // ...
}).then(res => {
    // ...
})

then方法可以手动return一个Promimse，后续的then方法会等待Promise成功才会继续执行：

new Promise(function (resolve, reject) {
    resolve()
}).then(res => {
    // 返回一个自定义的Promise对象
    return new Promise(function (resolve, reject) {
        settimeout(function () {
            reslove('成功')
        }, 3000)
    })
}).then(res => {
    console.log(res) // 3秒后，输出“成功”
})

# 异常处理

Promise对象有两种方式捕获失败状态：

  1. 使用then方法的第二个参数：
new Promise(function (resolve, reject) {
    reject('Promise失败') // 失败
}).then(
    function(resolve, reject) {},
    function(err) {
        console.log(err) // 输出“Promise失败”
    }
)

  1. 使用catch方法

该方法会捕获Promise链条中任意位置的错误

new Promise(function (resolve, reject) {
    reject('错误')
}).then(res => {
    // ...
}).then(res => {
    // ...
}).then(res => {
    // ...
}).catch(err) {
    console.log(err) // 输出“错误”
}

# 执行时序

JS中的任务大致分为同步任务和异步任务两种，而异步任务又分为宏任务和微任务两种。

Promise属于微任务，其执行优先级高于宏任务。

# async/await

Promise拥有一种语法糖，可以让我们使用同步编程的方式编写异步代码，彻底从回调函数中解脱出来。他就是async/await:

// 在函数开头生命async
async function () {
    // 在Promise对象前使用await，代码将等待Promise状态变更后再继续执行
    const res = await new Promise((resolve, rejcet) => {resolve('成功')})
    console.log(res) // 成功
}
